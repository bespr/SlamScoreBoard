Slam Score Board

Slam Score Board (SSB) ist das Resultate-Projektionstool für Poetry Slam Veranstaltungen.

Mit SSB ist die Projektion der Poetry Slam Resultate ein Kinderspiel. Die Projektion der Resultate kann individuell angepasst werden (wie viele Runden, welche Jury, etc.). Die Bedienung erfolgt einfach über einen Webbrowser. Dadurch müssen Sie keine zusätzlichen Anwendungen installieren sondern nur ein SSB Verzeichnis erstellen und darin die Webseite-Daten speichern.

Das Wichtigste in Kürze:

  • Die gängigsten Wettbewerbmodi können eingestellt werden (z.B. Vorrunden-Finale, Cup-Modus, etc.).
  • Jurywertung kann angepasst werden (z.B. Noten mit einer Dezimalstelle, Streichnoten, Applaus, etc.).
  • Einstellungen können vorab gespeichert und dann kurz vor der Veranstaltung wieder geladen werden.
  • Exportieren der Resultate in einer html- oder csv-Tabelle.
  • SSB läuft auf den gängisten Browsern (Firefox, Chrome, Internet Explorer).
